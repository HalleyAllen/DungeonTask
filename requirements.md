# DungeonTask 需求文档

> 最后更新：2026-05-19

## 项目概述

DungeonTask 是一款面向鸿蒙（HarmonyOS）平台的日常记录与自我管理应用，核心功能包括：任务清单、健身计划、句子收集、词语收集、目标/愿望清单、备忘录六个模块。

---

## 功能需求

### 1. 任务清单（每日工作记录）

- 支持按日期记录每日工作任务
- 任务支持：添加、编辑、删除、标记完成
- 任务列表按日期分组展示（今日 / 历史）
- 支持任务完成状态打卡
- **数据本地持久化存储**
- **支持数据导出**

### 2. 健身计划

- 支持设置健身计划规律（如：每周一/三/五练胸、背、腿等）
- 计划按周期排布，支持自定义训练项目
- 每次训练完成后可打卡标记
- 展示健身计划日历 / 列表视图
- 可查看历史打卡记录
- **数据本地持久化存储**
- **支持数据导出**

### 3. 句子收集（有意思的句子）

- 支持添加、编辑、删除句子
- 句子列表展示，支持快速浏览
- 支持按标签或分类查看（可选）
- **数据本地持久化存储**
- **支持数据导出**

### 4. 词语收集

- 支持添加、编辑、删除词语
- 每个词语记录：词语名称、拼音、释义
- 词语列表展示，支持快速浏览
- **数据本地持久化存储**
- **支持数据导出**

### 5. 目标 / 愿望清单

- 支持设置短期 / 长期目标
- 每个目标支持标记完成进度（百分比或步骤）
- 支持添加、编辑、删除目标
- 目标列表展示，已完成/进行中分类查看
- **数据本地持久化存储**
- **支持数据导出**

### 6. 备忘录 / 随手记

- 快速记录临时想法，不归类到具体模块
- 支持添加、编辑、删除备忘录
- 备忘录列表按时间倒序展示（最新的在最前）
- 支持按关键词搜索备忘录内容
- **数据本地持久化存储**
- **支持数据导出**

### 7. 数据统计 / 回顾（可选，后续迭代）

- 本周完成任务数统计
- 健身打卡天数统计
- 目标完成进度总览
- 历史句子 / 词语浏览记录

---

## 非功能需求

| 项目 | 说明 |
|------|------|
| 平台 | HarmonyOS（ArkTS / ArkUI） |
| 数据存储 | 本地持久化（Preferences 或 关系型数据库 RDB） |
| 数据导出 | 支持将各模块数据导出为文件（如 JSON / CSV） |
| 提醒功能 | 本次暂不考虑 |
| UI 风格 | 简洁、易用，适合日常快速记录 |

---

## 待确认 / 可扩展功能建议

以下功能为可选扩展方向，可在后续迭代中考虑：

1. **搜索功能**：在各模块中支持关键词搜索
2. **标签 / 分类**：为任务、句子、词语、备忘录添加自定义标签，便于筛选
3. **数据导入**：从导出文件中恢复数据
4. **主题切换**：浅色 / 深色模式
5. **备份到云端**：后续可考虑鸿蒙分布式数据管理或云存储
6. ** widget 桌面小组件**：快速查看今日任务或随机展示句子
7. **打卡激励**：连续打卡天数、成就徽章等 gamification 元素
8. **习惯打卡**：通用习惯养成模块（如每天喝水、早睡、阅读）
9. **倒计时 / 重要日期**：记录生日、纪念日等，首页展示倒计时
